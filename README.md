1. Project Overview
A Python based app designed for users to track expenses, categorize spending, and calculate balances.
The project aims to simplify budgeting costs for people.

3. Features
User Management: Track expenses of user 
Expense Entry: Add new expenses with amount, category, and type.
Expense Removal: choose what expense category to delete from, choose what item to delete
Data Storage: Save, append, and read expense data.
View History: Display all recorded expenses.
Summary & Calculation:
Total expenditure per category.
Total amount spent.

5. Technologies Used
Language: Python 3.11.9 

6. Installation and Setup
Clone the repository: git clone [repository_url]
Navigate to project folder: cd [Expense_racker9000]
Ensure Python is installed (python 3.11.9).
Run the application: The Final Result with comments.py 

7. Usage Guide
Add Expense: Choose option to input amount, and enter description.
View Expense: View a formatted table of expenses in category.
Total Expense: Run the summary function to see net balance. 
Remove Expense, remove an item from select category.
